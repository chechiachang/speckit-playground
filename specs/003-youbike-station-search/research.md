# Research: 台北市 YouBike 2.0 即時站點查詢器

## Decisions

- Use the Taipei YouBike 2.0 public JSON station feed as the source of station data.
- Keep the UI single-page and list-based.
- Search should match both station name and address keywords.
- Show loading, empty, and error states.

## Notes

- No extra scope such as maps, routing, or favorites.
- Keep behavior simple and user-facing.
